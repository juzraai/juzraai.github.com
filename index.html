My projects

CORDIS Projects Crawler (currently developing)

This app/API can fetch information of all CORDIS projects. There's a CLI for users, and there are handy methods for developers! :) It can download project pages and publication list JSON strings, extract information them, export data to CSV, ... and see the feature list! :D

Crawljax Plugins (development paused)

My useful Crawljax Plugins. Currently there's only one: SaveHTML which helps you to save every DOM state to file. Filename and directory name will be created just as they are in the URL, so you can create a site mirror with Crawljax and my SaveHTML plugin.

Check out my URL2File class too, it can be useful in other cases too.

HeritrixRemote (development paused)

I'm currently working on this application which is needed at work. The aim is to simplify controlling Heritrix 3.x crawler from the command line especially when you've got a lot of crawling jobs to manage.

PubSearch (abandoned after 1.1)

This was my thesis at university. It crawls websites of publication databases, fetches publications of the given author and stores them in a database. I stopped development at version 1.1 because it's not flexible enough to crawl any kinds of sites, so I moved on to PubSearch 2.

PubSearch 2 (development paused)

The crawling engines are now outside of the application, in 3rd party libraries, and PubSearch 2 will communicate with them thru an interface. PubSearch 1.1 crawler engine will be transformed to use that interface and bundled with PubSearch 2, so old crawler definition files for PubSearch 1.1 will remain useable.

The project was planned, started, but my time and motivation flew away somehow :S ... but maybe I'll continue it someday.
